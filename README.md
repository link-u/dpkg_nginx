# How to build

## prework

```
git submodule update --init --recursive
```

- [Download Nginx tarball (source code)]("http://nginx.org/en/download.html")

```bash
tar xvf nginx-*.tar.gz
cd nginx-*/
dh_make --createorig --single
```

## build!

```bash
bash all.sh
```
